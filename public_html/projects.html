At the National Center for Supercomputing Applications (NCSA), we develop clever scientific web applications and tools to interact, analyze and visualize data from astronomical surveys to speed the scientific discovery process.

Current ongoing projects

desaccess development

This project focus on desaccess, a set of tools to access and analyze astronomical data, This web application is open sourced and is the main tool used by hundreds of astronomer around the Globe. This web application provides means to query a Oralce DB and generate cutouts from astronomical imges. This project has 3 main components: Front-end, back-end and deployment. These 3 areas are being developed and we have a few extra features we would like to add. For a longer description of these tools plese check here https://des.ncsa.illinois.edu/releases/dr1/dr1-access/dr1-ncsa.

DESaccess code as well as the release page are open sourced. The frontend was developed using web components, HTML, JS and Polymer which allows the re-use of already existing elements. The backend is mainly developed using the Python Tornado web framework. The submission jobs are handled by Celery which is a distributed queue manager written in Python. All of these applications are containerized using Docker and all the nodes resources and scheduling, as well as all the deployment, is managed by Kubernetes which allows to run, manage and scale containerized applications in a robust and efficient manner.

Galaxy images recommendation system

This project will create a recommendation system when visualizing galaxy images and make recommendations to the user on similar galaxies. The main idea is to run a Convolutional Neural Network (CNN) and/or autoencoders to rank galaxies from a large DB by their similarities to a query image. There is already work done for this projects and mentor will provide all the necessary images and tools to be used. Ideally this will end up in a web application itself or as a addition to desaccesss.
Requirements for back-end:
Python, Tensorflow, Scikit-learn, image manipulation, Machine learning

Requirements for front-end:
Polymer, HTML, Javascript, Python, Tornado

Difficulty:
medium

Somsphere

This project will convert current Self-Organizing-Map in Spherical Coordinates written in Python to a C++ implementation for efficiency. The application of Self-Organized-Maps, a class of unsupervised machine learning algorithm are vast. From clustering to accurate prediction and labeling. This is a on-going project with multiple applications to real scientific problems.

data vista

This project aims to provide an easy-to-use online plotting tool, for the tabular data retrieved from the DES database. The main usability will be to quickly assess the results of the retrieved queries by selecting between 2 to 4 columns to plot (the third and fourth columns will represent color and marker size). For refined visualizations we have more elaborated tools as Jupyter notebooks, then this feature needs to be robust, fast, simple, and clean.
Requirements:
Python, Bokeh, Holoviews, Polymer

easy

easyaccess plugin features

This project will add a plugin capabilities to easyaccess, a enhanced command line client to access data from astronomical surveys, currently used by hundreds of scientists, we have plans to make the client compatible with DES public access. It is entired written in python, it is python2/3 compatible and it has a long number of nice and customized features, like tab-completion, find for user/tables download and upload tables with a large range of data file format. The idea for this project is to add a functionality os other scientists can add plugins and features.
